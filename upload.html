This innovative system allows you to navigate and interact with the exam interface using voice commands.
Below are the commands and instructions for using the portal effectively:

1. Getting Started:

  • Upon entering the exam page, the recording will start automatically.
  • Voice commands are activated by simply saying them aloud.
  • Speak clearly and at a moderate pace.

2. Voice Commands:

  • Select Question (number):
  • Usage: Select the question number you want to answer.
  • Example: "Select Question 3" will move the cursor to Question 3.

  • Stop over Finish:
  • Usage: Stop answering a question and finish recording.
  • Example: Saying "Stop" or "Finish" will stop answering and finish recording for the current question..

  • Edit Question (number):
  • Usage: Edit the answer for a specific question.
  • Example: "Edit Question 5" will enter edit mode for Question 5.

  • Delete All:
  • Usage: Delete the entire answer for a particular question.
  • Example: Saying "Delete All" will delete the whole answer for the current question.

  • Move cursor next:
  • Usage: while in edit mode, cursor moves to the next sentence
  • Example:On saying "Move Cursor Next", if the cursor is at the beginning of the first sentence, the cursor will be moved to the beginning of the next sentence.

  • Move cursor back:
  • Usage: While in edit mode, cursor moves to the previous sentence.
  • Example:On Saying "Move cursor back", if the cursor is at the beginning of the second sentence, the cursor will be moved back to the beginning of the previous sentence.

  • Select This Sentence:
  • Usage: While in edit mode, It Chooses the sentence where the cursor is located.
  • Example:If the cursor is located in the 1st sentence, then upon saying "Select This Sentence" will select the 1st sentence.

  • Highlight Text:
  • Usage: Highlight a specific portion of text.
  • Example: On Saying "Highlight Text" will Bold the selected text.

  • Delete Sentence:
  • Usage: Delete the sentence where the cursor is located.
  • Example: Saying "Delete Sentence" will delete the sentence where the cursor is located.

  • Add text:
  • Usage: Add text to the answer for a specific question.
  • Example: If you say "Add Text" and then say "This is a test", the text "This is a test" will be added to the answer where cursor is located.

  • Save and Exit:
  • Usage: Save the changes and exit the edit mode.
  • Example: Saying "Save and Exit" will save the changes and exit the edit mode.

  • Speak button:
  • Usage: The question is read out loud by the program.
  • Example: Clicking on the "Speak" button will read out the question.

  • Read Text button:
  • Usage: The answer is read out loud by the program.
  • Example: Clicking on the "Read Text" button will read out the answer.
